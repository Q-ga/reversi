# CONTEXT — リバーシ 用語集

このプロジェクトで使う言葉の定義。実装の詳細は書かない（用語の意味だけ）。

## プレイヤー / 識別

- **プロフィール**：名前だけを持つ登録済みの人。上限2人。戦績の個人集計の対象。
- **ゲスト**：その対局限りの一時的な人。プロフィール枠を消費せず、個人集計の対象にもしない。
- **CPU**：コンピュータの対戦相手。難易度（かんたん/ふつう/つよい）を持つ。
- **主役（メインプレイヤー）**：1対局につき1人指定する「BGM・演出の基準となる人」。優勢/劣勢の判定や演出は常にこの人の視点で行う。盤上では黒/白いずれかの色を担う。

## 対局

- **先攻 / 後攻**：先攻＝黒、後攻＝白。黒が先に打つ。
- **手番**：今打つ番の色。
- **パス**：打てる場所が無い手番が、着手せず相手に番を渡すこと。
- **待った**：直前の局面に戻す操作。回数制限なし。
- **棋譜**：1対局の全手順を標準表記（列 a–h／行 1–8、パスは ps）で並べた記録。
- **連鎖めくり**：着手で返る石を、一度に変えず置いた石から外側へ波状に1枚ずつ裏返す見せ方。

## 形勢・演出

- **形勢**：主役視点での有利不利。「優勢／互角／劣勢」の3段階で表す。序中盤は盤上の位置価値、終盤は石数差を重視して判定する（ハイブリッド）。
- **スポット演出**：特定の瞬間に出す光と音の演出。対象＝大量返し／角取り／逆転／終局／完封／パス／ラス1マス。浮遊テキストは出さない（勝敗などの文字情報は操作画面側で表示する）。

## 記録の可否

- 登録ユーザー（プロフィール）が1人でも対局に参加していれば、その対局を記録する。
- ゲスト同士、および登録ユーザーが居ない組み合わせ（ゲスト対CPU等）は記録しない。
